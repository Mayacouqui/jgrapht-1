JGraphT 


About JGraphT

JGraphT is a free Java class library that provides mathematical graph-theory objects and algorithms. JGraphT supports various types of graphs including:

  • directed and undirected graphs.
  • graphs with weighted / unweighted / labeled or any user-defined edges.
  • various edge multiplicity options, including: simple-graphs, multigraphs, pseudographs.
  • unmodifiable graphs - allow modules to provide "read-only" access to internal graphs.
  • listenable graphs - allow external listeners to track modification events.
  • subgraphs graphs that are auto-updating subgraph views on other graphs.
  • all compositions of above graphs.

Although powerful, the JGraphT API is designed to be simple. For example, graph vertices can be of any objects. This allows you to create graphs based on: Strings, URLs, XML documents, etc; you can even create graphs of graphs! This code example shows how it works.

Other features offered by JGraphT:

  • complete source code, under the terms of the GNU Lesser General Public License.
  • comprehensive Javadocs.
  • easy extensibility.
     

JGraphT vs. JGraph

JGraphT and JGraph are two different libraries which are optimized for different purposes.

JGraphT:

  • is optimized for algorithms.
  • has a clean and simple API that is designed for a safe "auto-complete programming".
  • is designed for high-performance and for large-scale graphs.
  • provides visualizations by using JGraph (try this demo).

JGraph:

  • is optimized for graph visualization and GUI-based editing.
  • has a complicated API with non-trivial casting; requires fluency with documentation.
  • is slower and consumes more resources.
  • currently does not use JGraphT.

You will probably find JGraphT easier to start with and more eligible for many applications. However, if your application is primarily graph visualization or GUI-based graph editing, you may the JGraphT benefits less compelling.
 

Download

You can download the latest version of JGraphT from the project page on SourceForge:


jgrapht-0.4.0.zip		 Windows Download
A ZIP file that contains source code and runtime files of JGraphT.

jgrapht-0.4.0.tar.gz 		Linux Download
A tar.gz file that contains source code and runtime files of JGraphT.

Note that there are two downloads, one for Windows-based systems, and one for Linux/Unix-based systems. The downloads contain identical Java source code and a README.html file to help you to get started.

The JavaDoc HTML files are available online. They are also included in the downloads in case you wish to view them locally.
 

Support

The JGraphT project is maintained at SourceForge, where you can find the following support facilities:

  • the feature tracker - use it to request new features for JGraphT.
  • the bug tracker - use it to report bugs in JGraphT.
  • the help forum - use it to post your JGraphT questions an to find answers.

If you are reporting a bug, please describe a reproducing scenario. If possible, attach a small program with a description of what is wrong and need of fixing.
