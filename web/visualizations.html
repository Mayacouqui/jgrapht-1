Demo Applet

The following applet shows how a JGraphT graph can be visualized using JGraph. Try to play and drag around the vertices and edges to get the feel of it.

Note: Java 1.3 or above must be installed for this applet to work correctly.


Java 2 Standard Edition v 1.3 or above is required for this applet.
Download the it from http://java.sun.com.

How it Works

It's very simple: starting from version 0.4.1 the JGraphT library comes with an adapter that makes JGraphT graphs compatible with JGraph. To visualize a JGraphT graph you just need to initialize JGraph via that adapter.

Example code:

     // create a JGraphT graph
     GraphFactory    gf = GraphFactory.getFactory(  );
     ListenableGraph g  = gf.createListenableGraph( gf.createSimpleGraph(  ) );

     // create a visualization using JGraph, via the adapter
      JGraph jgraph = new JGraph( new JGraphModelAdapter( g ) );

Is that all?!  Yes, that's all. Any modification now made to the graph g will automatically be reflected by the JGraph component.

 

Source Code of the Applet

The source code of this demo applet ships with JGraphT 0.4.1 and above. The full class name is: org._3pq.jgrapht.demo.JGraphAdapterDemo. You can either download JGraphT or view it directly from CVS.
