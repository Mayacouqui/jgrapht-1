Introduction

JGraphT is a free Java class library that provides mathematical graph-theory objects and algorithms. It runs on Java 2 Platform (JDK 1.3 or later).

Getting Started

The package org._3pq.jgrapht.demo includes a little demo application to help you get started. If you spawn a little demo of your own and think others can use it, please send it to me and I will add it as another demo to the package.

Documentation

A local copy of the Javadoc HTML files is included in this distribution. The latest version of these files is also available on-line.

Dependencies

  • JGraphT requires JDK 1.3 or later. You might get JGraphT working with JDK 1.2.x, but I haven't tested this. JGraphT certainly won't work with JDK 1.1.x because it requires the collection framework that was only introduced in JDK 1.2.
     
  • JUnit is a unit testing framework. You need JUnit only if you run the unit tests, or if you compile the library from its source code. The junit.jar runtime file is included in the distribution. JUnit is licensed under the terms of the IBM Common Public License. You can find out more about JUnit and/or download the latest version from http://www.junit.org. The JUnit tests included with JGraphT have been created using JUnit 3.8.1.
